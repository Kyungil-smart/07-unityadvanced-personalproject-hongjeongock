# 🧟 마을 수호자

---

## 🎮 게임 소개

**좀비 서바이벌**은 3D 탑다운 뷰 방식의 생존 게임입니다.  
플레이어는 좀비의 공격을 막아내는 동시에, 자원을 수집하여 거점을 강화해야 합니다.

---

## 🕹️ 조작법

| 키 | 동작 |
|:---:|:---|
| `W` `A` `S` `D` | 캐릭터 이동 |
| `마우스 좌클릭` | 공격 |
| `마우스 휠` | 줌 인 / 줌 아웃 |
| `E` | 상호작용 |
| `ESC` | 일시정지|

---

## 🏠 주요 시스템

### ⚔️ 전투
- 마우스 좌클릭으로 주변 좀비를 공격합니다.
- 공격에는 쿨타임이 존재하며, 좀비 공격 시 체력이 감소합니다.
- 체력이 0이 되면 게임 오버입니다.

### 🏚️ 집 업그레이드
- 집 내부에 들어가면 `Tap키`를 눌러 업그레이드를 할 수 있습니다.
- 업그레이드를 통해 거점을 강화할 수 있습니다.(미구현 : 타워 강화)
- 업그레이드에는 수집한 코인/자원이 필요합니다.

### 🎒 아이템 & 코인 수집
- 맵에 떨어진 아이템과 코인에 접근하면 자동으로 수집됩니다.
- 코인과 아이템은 업그레이드 비용으로 사용됩니다.
- 핫바가 있지만 아직 무기를 구현 못했습니다...(추가예정)

---

## 💀 게임 오버

체력이 0이 되면 게임 오버 화면이 표시됩니다.  
게임 오버 화면에서 재시작하거나 메인 메뉴로 돌아갈 수 있습니다.
